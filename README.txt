This is a simple Facebook messenger chat bot that provides services including 
surfing Reddit to provide users with different posts depending on their selection, 
using OpenWeatherMap’s API to provide current weather data, and a third party 
RESTful API to provide users with different soccer standings and daily fixtures 
of different leagues.

Commands:
-using quick replies list
-weather
-fixtures + 
	-Premier League
	-Ligue 1
	-Bundesliga
	-Serie A
-standings + 
	-Premier League
	-Ligue 1
	-Bundesliga
	-Serie A
